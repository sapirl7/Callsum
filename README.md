Сервис анализа и саммаризации звонков
Этот сервис предназначен для автоматической обработки аудиозаписей звонков. Он выполняет транскрибацию, определяет спикеров и создает структурированную выжимку по ключевым блокам (технический, коммерческий, прочее).

Технологический стек
Docker & Docker Compose: для контейнеризации и запуска сервиса.

Python: язык разработки.

Faster-Whisper: для транскрипции аудио в текст.

Pyannote.audio: для диаризации (определения спикеров).

Ollama (Llama3): для генерации структурированного саммари.

Быстрый старт
1. Предварительные требования
Для работы сервиса требуется токен доступа Hugging Face.

ВАЖНОЕ ЗАМЕЧАНИЕ О КОНФИДЕНЦИАЛЬНОСТИ:
Токен используется ИСКЛЮЧИТЕЛЬНО для однократного скачивания open-source модели pyannote/speaker-diarization с репозитория Hugging Face при первом запуске. Это требование создателей модели.

Никакие данные, включая аудиофайлы, тексты звонков или ваши личные данные, НИКОГДА не отправляются на серверы Hugging Face или куда-либо еще. Весь процесс обработки аудио на 100% происходит локально внутри Docker-контейнера.

2. Установка
Получите свой токен на huggingface.co/settings/tokens.

Создайте файл конфигурации. Скопируйте файл .env.example и переименуйте его в .env.

Вставьте ваш токен в файл .env:

HF_TOKEN=ваш_токен_здесь

Поместите аудиофайл для обработки в корень проекта под именем sample_audio.wav.

Запустите сервис командой:

docker-compose up --build

При первом запуске скачивание моделей может занять 10-20 минут.

3. Результат
После успешной обработки результат будет сохранен в файл structured_summary.json.
